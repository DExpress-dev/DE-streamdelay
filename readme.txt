一：可执行文件delay和log.json需放在同一文件夹下，其他配置文件不需要。
    log.json用于指导log的存放路径。
二：手动执行delay的例子如下：

./delay  http://gsports.livebak.ali.otvcloud.com/otv/wxty/live/channel01/1300.m3u8  /mnt/hgfs/D/go_language/go_streamcheck/src/streamcheck/source1/

http://gsports.livebak.ali.otvcloud.com/otv/wxty/live/channel01/1300.m3u8  指定源流的地址， 
/mnt/hgfs/D/go_language/go_streamcheck/src/streamcheck/source1/ 指定下载下来的流存放位置。
